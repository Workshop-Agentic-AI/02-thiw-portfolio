# 02-thiw-portfolio

Personal portfolio project for Workshop Agentic AI.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Workshop-Agentic-AI/02-thiw-portfolio.git
cd 02-thiw-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
